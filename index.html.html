Koneksi Google Sheets
Cara Setup Google Sheets
Data checklist akan otomatis masuk ke Google Sheets setiap kali laporan dikirim. Ikuti langkah berikut:
1
Buka Google Sheets, buat spreadsheet baru. Beri nama misalnya MEP Checklist GUNA JASA.
2
Di spreadsheet tersebut, klik menu Extensions → Apps Script.
3
Hapus semua kode yang ada, lalu paste kode berikut:
function doPost(e) { try { var data = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.getActiveSpreadsheet(); // Sheet 1: Ringkasan Laporan var sh1 = ss.getSheetByName('Laporan') || ss.insertSheet('Laporan'); if (sh1.getLastRow() === 0) { sh1.appendRow(['ID','Tanggal','Proyek','Pelapor','Zona','Jumlah Item','OK','NOK','Waktu Kirim']); sh1.getRange(1,1,1,9).setFontWeight('bold').setBackground('#185FA5').setFontColor('#ffffff'); } var ok = data.items.filter(function(i){return i.status==='ok'}).length; var nok = data.items.filter(function(i){return i.status==='nok'}).length; sh1.appendRow([data.id, data.date, data.project, data.reporter, data.zone||'-', data.items.length, ok, nok, new Date().toLocaleString('id-ID')]); // Sheet 2: Detail Item var sh2 = ss.getSheetByName('Detail Item') || ss.insertSheet('Detail Item'); if (sh2.getLastRow() === 0) { sh2.appendRow(['ID Laporan','Tanggal','Proyek','Pelapor','Kategori','Area','Deskripsi','Status','Jumlah Foto','URL Foto']); sh2.getRange(1,1,1,10).setFontWeight('bold').setBackground('#185FA5').setFontColor('#ffffff'); } data.items.forEach(function(item) { sh2.appendRow([data.id, data.date, data.project, data.reporter, item.category||'-', item.area||'-', item.desc||'-', item.status, item.photoCount||0, item.photoUrls||'-']); }); return ContentService.createTextOutput(JSON.stringify({status:'ok'})).setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService.createTextOutput(JSON.stringify({status:'error',msg:err.toString()})).setMimeType(ContentService.MimeType.JSON); } }
4
Klik Deploy → New deployment. Pilih type Web app. Set Execute as: Me, Who has access: Anyone. Klik Deploy dan izinkan akses.
5
Copy URL Web App yang muncul (dimulai dengan https://script.google.com/macros/s/...), lalu paste di kotak bawah ini.

Cara Deploy ke Web (Gratis)
File index.html ini bisa dihosting gratis di:

1. GitHub Pages — Upload ke GitHub repo, aktifkan Pages di Settings.
2. Netlify — Drag & drop folder ke netlify.com/drop.
3. Google Drive — Upload, share as "Anyone with link", buka sebagai Google Sites.

Setelah online, bagikan link ke seluruh tim lapangan. Bisa disimpan sebagai shortcut di Home Screen HP.
